# Árbol de Descomposición Funcional
## Módulo: Gestión de Ventas


```text
Gestión de Ventas
│
├── Registro de Venta
│   ├── Seleccionar cliente
│   ├── Seleccionar producto
│   ├── Validar disponibilidad
│   ├── Calcular total
│   └── Confirmar venta
│
├── Facturación
│   ├── Generar factura
│   ├── Asignar número de factura
│   ├── Calcular impuestos
│   └── Imprimir o exportar factura
│
├── Historial de Ventas
│   ├── Consultar ventas por fecha
│   ├── Consultar ventas por cliente
│   └── Consultar ventas por producto
│
└── Reportes de Ventas
    ├── Reporte diario
    ├── Reporte mensual
    └── Reporte anual
```
