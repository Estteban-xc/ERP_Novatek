# Product Backlog

## Historias de Usuario

### HU-01 – Registrar producto
Como administrador, quiero registrar productos con su información y código IMEI para mantener control del inventario.

### HU-02 – Registrar cliente
Como administrador, quiero registrar clientes con sus datos básicos para asociarlos a ventas.

### HU-03 – Realizar venta
Como usuario, quiero registrar ventas seleccionando productos y cliente para llevar control comercial.

### HU-04 – Consultar inventario
Como usuario, quiero consultar el inventario disponible para conocer el stock actual.

### HU-05 – Generar reporte básico
Como administrador, quiero visualizar un reporte simple de ventas para analizar la actividad del negocio.

---

## Priorización inicial

1. HU-01 Registrar producto  
2. HU-02 Registrar cliente  
3. HU-04 Consultar inventario  
4. HU-03 Realizar venta  
5. HU-05 Generar reporte
