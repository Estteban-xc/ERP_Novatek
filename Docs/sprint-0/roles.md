# Roles del Proyecto

## Product Owner
Responsable de definir los requerimientos del sistema, priorizar el backlog y validar que el producto cumpla las necesidades del proyecto académico.

## Scrum Master
Responsable de asegurar el cumplimiento de la metodología Scrum, organizar el trabajo del Sprint y facilitar la coordinación del equipo.

## Equipo de Desarrollo
Responsables de:

- Diseñar la arquitectura del sistema  
- Configurar el entorno de desarrollo  
- Crear la base de datos  
- Implementar los módulos del ERP  
- Mantener actualizado el repositorio  

Todos los integrantes participan activamente en el desarrollo y documentación del sistema.
