# Visión del Proyecto

## Problema identificado
Las pequeñas y medianas empresas que comercializan productos tecnológicos suelen gestionar inventario, ventas y clientes mediante hojas de cálculo o sistemas no integrados, lo que genera:

- Errores en el control de stock  
- Dificultad para rastrear productos por IMEI  
- Falta de reportes consolidados  
- Procesos manuales lentos  

## Objetivo del sistema
Desarrollar un sistema ERP de escritorio que permita gestionar de manera integrada:

- Inventario de productos  
- Clientes y proveedores  
- Registro de ventas  
- Control administrativo básico  

## Alcance del sistema
El sistema incluirá inicialmente:

- Registro y consulta de productos  
- Registro de clientes  
- Registro de ventas  
- Consulta de inventario  
- Base de datos estructurada en MySQL  

## Módulos principales

1. Módulo de Inventario  
2. Módulo de Clientes  
3. Módulo de Ventas  
4. Dashboard principal  

## Resultado esperado
Contar con una aplicación de escritorio funcional, modular y escalable, que sirva como base para futuras ampliaciones del ERP.
